Comandos do git

Primeiro a se fazer é acessar a pasta pelo terminal. git init: inicializar um repositório git . git add REDME.md: seria um texto explicativo sobre o projeto. git add . : adicionar todos os arquivos do projeto. git commit –m “first commit” : adicionando o nome. git branch –M main: convenção do github. git remote add origin git@github.com:front-leinners/... : máquina local a máquina remota. git push –u origin main: enviando para o github realmente
